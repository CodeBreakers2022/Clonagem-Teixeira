## Cloning - Teixeira Turismo

This project was developed as part of the Systems Analysis and Design module, with the objective of improving our front-end development skills and deepening our knowledge in business rule analysis, requirement analysis, as well as functional and non-functional requirements.

<br>

<h2 align="center">
    How to run the project:
</h2>

<br>

1. Download XAMPP and place the code in the "C:\xampp\htdocs" folder.
2. Create the database using the script available in the code in the data folder.
3. Run the application on some online server, for example XAMPP (start the Apache service, and the Mysql).
4. Open the browser and access: `http://localhost:3000`
